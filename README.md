# Secure Web-Based Coding Assessment System  
### Using Compiler Design Concepts

## 📌 Project Overview
This project is a web-based coding assessment platform designed for placement-style tests.  
It integrates compiler design concepts such as lexical, syntax, and semantic analysis.

## 🎯 Objectives
- Design a secure coding environment  
- Implement lexical, syntax, and semantic analysis  
- Validate code correctness  

## ⚙️ Features
- Code editor  
- Timer  
- Lexical, syntax, semantic analysis  
- Tab-switch warning  

## 🏗️ Project Structure
secure-code-evaluator/
├── compiler/
├── frontend/
└── README.md

## 🚀 How to Run
Compile C files:
gcc lexer.c -o lexer
gcc syntax.c -o syntax
gcc semantic.c -o semantic

Run:
./lexer
./syntax
./semantic

Open frontend/index.html in browser.
